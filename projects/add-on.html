Blender AddOn - Trajectory Interpolation

Teamwork project of Animation subject (Year 3 Multimedia Engineering) at ETSE (Universitat de València).

The user must create a set of keyframes to interpolate between them and select the type of interpolation desired (linear, Catmull-Rom splines or Hermite). For each interpolation type specific parameters can be changed. 

The object faces orientation can be selected (see example video where a car model is forced to always face forward as in real life).

Authors: Darío Rodríguez Hernández, Ana Gloria Gálvez Mellado, Bekzat Mukhamedali and Guillem Barceló Orts.

This addon code is public and can be found on this link


Also, the user can add an oscilation to achieve a more natural effect by adding a wrapping angle value on a keyframe. This video shows the results of adding different values of wrapping angles at certain keyframes to achieve the skidding effect.
