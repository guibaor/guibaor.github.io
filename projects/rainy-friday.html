Rainy Friday - Game Prototype


Build & game design document are public and can be found on this link

Rainy Friday was a videogame prototype created from scratch from the game design document to the build by Ana Gloria Gálvez Mellado and me as a part of the Game Development subject (Masarykova Univerzita).

All the selling points, target, game main features, art-style and time constraints were studied at the beginning on a game design document .

Several mechanics were implemented in the game. The gameplay goal was to give the player freedom to explore the scene and talk to the NPCs. Those would give them quests as mini-games that, once accomplished, would unlock new quests and make the game progress.

Main features implemented were:

  • A dialogue system: Using YarnSpinner (based on nodes) for an easy narrative modification.
  • Inventory and pickable objects: Dynamic inventory with object descriptions.
  • In-game music player: With a song-unlocking system when the player finished different quests.
  • Escaping the maze mini-game: Using Unity NavMesh Agents.
  • Retro space shooter mini-game: With our own pixel-art sprites.
  • Quick time event mini-game: Implementing the quick time event system
