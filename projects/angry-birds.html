Angry Birds recreation using Processing


Simulation of a level of the famous Rovio Entertainment game using the "Física" Processing physics engine created by Ricard Marxer.

The launch of the bird has been implemented using an elastic joint that only acts on a threshold of the X axis, and when it exceeds it ,the elastic joint breaks, throwing the bird. Collision with green enemies is calculated by checking if they have collided with any of the neighboring objects.

A scoring system has also been implemented, as well as hostile elements such as the hanging saws that cause the player to lose a life if the bird collides with them.
