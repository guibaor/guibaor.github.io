Numerical Integration Methods

Interactive simulation of an elastic pendulum.

The movement is calculated with different integration methods: Explicit Euler, Semi-explicit Euler, Heun, Runge-Kutta 2 and Runge-Kutta 4, that can be switched at simulation time through the keyboard. The pendulum movement is achieved by adding an initial velocity to the spring together with the force of gravity

By clicking on the screen the position of the ball that is attached to the spring can be changed. The screen shows the information related to the integration data and the keyboard controls.

Interactive simulation of a parabolic shot with variable height and bounce. 

The movement is calculated with different integration methods: Explicit Euler, Semi-explicit Euler, Heun, Runge-Kutta 2 and Runge-Kutta 4, that can be changed at simulation time through the keyboard.

The initial launch speed of the ball can be changed by clicking (the speed vector that will be applied when starting the simulation when pressing spacebar is displayed). The screen shows the information related to the integration data, the value of the variables and the keyboard controls.
