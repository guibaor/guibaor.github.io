Pathfinding - A* algorithm & heuristic comparision

Unity application that shows how the A* algorithm works applied to pathfinding.

To start the A* search, the user must select the path destination by clicking on maze cell. Search is slowed down to see how A* works by filling the cells with different colors depending on their state in the search:

  • Green cells: Open set nodes (The cell is being explored).
  • Red cells: Closed set nodes (The cell has been totally explored).
  • Blue cells: The cells belongs to the final path set.

Also, the user can choose between Euler distance as heuristic or null heuristic. The video includes a comparison of the same search performed with Euler distance heuristic vs null heuristic.
